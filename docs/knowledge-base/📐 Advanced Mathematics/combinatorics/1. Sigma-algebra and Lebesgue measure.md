## 1. Sigma-Algebra 
For the source of definition, it's from wiki [^2], as is given in <span class="equation-citator-citation equation-citator-cite-callout" data-ec-kind="def" data-ec-refs='[{&quot;file&quot;:null,&quot;crossFile&quot;:null,&quot;tag&quot;:&quot;1.1&quot;}]' style="cursor: default; color: #4199df;">Definition. <span style="text-decoration: none; cursor: pointer; color: #4199df;">1.1</span></span>, The sigma-algebra **==describe the property that a set is always self-closed==**. 

> [!def:1.1|green] Sigma-algebra 
> If $X = {a, b, c, d}$, then an example of $\sigma$-algebra on $X$ is $Σ = \{ ∅, \{a, b\}, \{c, d\}, \{a, b, c, d\} \}$, where $\emptyset$ is the empty set. 
> 
> In general, a **finite algebra is always a σ-algebra** if it : 
> 1. Including whole set $(X\in \Sigma)$
> 2. Close for complementary set $(\text{if } A\in  \Sigma ,\text{then } X\backslash A \in  \Sigma  )$ 
> 3. Close for and $\text{if } A_{1}, A_{2}, \dots  \in \Sigma$, then $\cup_{i =1}^{\infty} A_{i} \in  \Sigma$ 

## 2. Lebesgue Measure  
Lebesgue measure defines the, see [^1] for details. 

the **Lebesgue measurable**  is the **standard way of assigning a [measure](https://en.wikipedia.org/wiki/Measure_\(mathematics\) "Measure (mathematics)") to [subsets](https://en.wikipedia.org/wiki/Subset "Subset") of [higher dimensional](https://en.wikipedia.org/wiki/Higher_dimension "Higher dimension") [Euclidean _n_-spaces](https://en.wikipedia.org/wiki/Euclidean_space "Euclidean space").** 
### (1) Length of the set 
we describe the **measure of  Lebesgue measurable set** $A$ as : 
$$
\lambda(A) \tag{2.1.1}
$$
for $I = [a, b]$ or $I = (a, b)$, we denote the length of set as $l(I)= b-a$ 

> [!def:lebsgue-outer-measure] Lebesgue Outer Measure 
> <b><mark style='background: transparent; color: red'>Lebesgue Outer Measure</mark></b>: For subset $E \subseteq R$ , we define the <b><mark style="background: transparent; color: orange"> Lebesgue outer measure</mark></b>  as the minimum discontinuous summation :
> $$
> \boxed{
> \lambda^{*} (E) = \text{inf} \left\{ \sum_{k =1}^{\infty} l(I_{k})  : (I_{k}) \text{is a sequence of open interval with } E \subset  \bigcup_{k = 1}^{\infty} I_{k} \right\}
> } \tag{lebesgue-measure}
> $$

> [!example]  
> We take $E = Q \cap [0, 1]$ as a 2-element set,  $I_{k}$ as a open set that contains each element of $E$, then we set the length of $I$ is : 
> $$l(I_{k}) = \frac{\epsilon}{2^{k}}$$
> Then the sum of length should be : 
> $$\sum_{k =1}^{\infty} l(I_{k}) = \epsilon$$
> then the $\lambda^{*} (E) \leq \epsilon$, i.e., we have $\lambda^{*} (E) = 0$

> 

The equivalent statement of  is, for any $\varepsilon > 0$, **there exists a covering $S$ of $\Omega$ with closed boxes**.  We note  <span class="equation-citator-citation equation-citator-cite-equation" data-ec-kind="eq" data-ec-refs='[{&quot;file&quot;:null,&quot;crossFile&quot;:null,&quot;tag&quot;:&quot;lebesgue-measure&quot;}]' style="cursor: default; color: #4199df;">(<span style="text-decoration: none; cursor: pointer; color: #4199df;">lebesgue-measure</span>)</span> can also be stated as : 
$$
m^{*} (\Omega) := \inf \left\{ \sigma(S) | S \text{ covering of } \Omega\right\} \tag{2.1.2}
$$
For a rectangular cuboid $C = I_{1 } \times \dots I_{n}$ with $\text{vol}(C) =  l(I_{1})\times l(I_{2}) \times \dots l(I_{n})$ 
$$
\lambda^*(E) = \inf \left\{ \sum_{k=1}^{\infty} \operatorname{vol}(C_k) : (C_k)_{k \in \mathbb{N}} \text{ is a sequence of products of open intervals with } E \subseteq \bigcup_{k=1}^{\infty} C_k \right\} \tag{2.1.3}
$$

> [!example] 
> This gives, the set and be convered with limited ranged boxes. **The classic example: a single point, then a finite set.** 
> Firstly, We Let $E = {0, 1, 2}$ (three points on the real line). Intuitively, three isolated points have "zero length." The outer measure confirms this. 
> For any ε > 0, cover each point with a tiny open interval:
> - 0 ∈ (−ε/6, ε/6), length = ε/3
> - 1 ∈ (1−ε/6, 1+ε/6), length = ε/3
> - 2 ∈ (2−ε/6, 2+ε/6), length = ε/3  
>   
>  For a range set  $E = [a, b]$ , Lebesgue measure is : 
>  $$\lambda^{*} ([a, b]) = \inf (a - \epsilon, b + \epsilon) = l([a,b]) = b -a$$

### (2) Useful properties 
1.  **Given any ==countable collection of sets== from an algebra (or σ‑algebra) $A$, we can replace it with a ==_disjoint_ countable collection== of sets that are also in $A$ and have the same union.**

**For every $\sigma-$ algebra $A$, We let $\{E_{n}\}_{n \geq 1}$ be an arbitrary sequence of a set $A$ (no-$\sigma$-algebra needed)**, then there exists a sequence of **pairwise disjoint** collection, $\{ F_{n} \}_{n}$, or a countable collection. satisfying for every $n$ : 
$$
\boxed{\bigcup_{k = 1}^{n} F_{k}  = \bigcup_{k = 1}^{n} E_{k}} \tag{2.2.1}
$$


To prove this, we can take the $F$ as following part : 
$$
F_{1} = G_{1} \qquad  F_{n} = \cup_{k = 1}^{n+1} E_{k} \backslash \cup_{k = 1}^{n} E_{k}  \quad  \to  \bigcup_{k = 1}^{n} F_{n} = \bigcup_{k =1}^{n} G_{n} \tag{2.2.2}
$$
so <b><mark style='background: transparent; color: orange'>any countable algebra can be represented as disjoint collections</mark></b>.  

### (3) Lebesgue measurable set 
***Caratheodory criterion*** : if a set $E$ has the following relation : for every $A \subset R^{n}$ 
$$\Large
\boxed{\lambda^{*} (E) =\lambda^{*}(A\cap E) + \lambda^{* }(A \cap  E^{c}) } \tag{2.3.1}
$$
where $E^c$ is the **complement set of $E$**,  We say it **satisfy the Caratheodory criterion**, or say it is <b><mark style='background: transparent; color: red'>Lebesgue measurable</mark></b>.  where $\lambda$ is the ***Lebesgue outer measure*** 

> [!note:2.1] Borel $\sigma$  algebra 
> We note Borel $\sigma$  algebra is the smallest $\sigma$ algebra containing all open sets. 

The **Lebesgue measure** of such set $E$ define as Lebesgue outer measure :
$$
\lambda(E) = \lambda^{*}(E) \tag{2.3.2}
$$

## 3. Axiom of Choice and Vitali Sets 

> [!axiom:axiom-of-choice] Axiom of Choice 
> This axiom states that : for every set $I$, and every $I$-indexed family $(S_{i})_{i \in  I}$ of <b><mark style='background: transparent; color: red'>non-empty sets</mark></b>, <b><mark style='background: transparent; color: orange'>there exists an</mark></b>  I-indexed set $(x_{i})_{i \in I}$ of elements of $\cup_{i \in I}$ such that $x_{i} \in  S_{i}$ for every $i\in I$
> 
> To be short, This axiom  says you can get at 1 element from $S_{i}$ each if its non-empty 

Vitali Sets[^3] is **not Lebesgue measurable**. A typical Vitali set is <b><mark style='background: transparent; color: orange'>to choose exactly one representative from the each equivalence class</mark></b> under the following relation: 
$$
x \sim y \qquad \text{iff} \quad x -y  \text{ is rational} \tag{3.1}
$$
or set the Vitali set as $V$ such that : 
$$
\text{if} \quad   u, v \in  V \qquad \text{and} \quad  v -u \in Q \quad \text{then } v = u \tag{3.2}
$$
from wiki, we know that we can find uncountable **pairwise-disjoint set** such that :  
$$V_{k} \in [-1, 2] \tag{3.3} $$
If $V_{k}$ is measurable, the **Lebesgue measure** for **pairwise-disjoint set** yields : 
$$
\lambda (\bigcup_{k = 1}^{\infty} V_{k}) = \sum_{k = 1} \lambda (V_{k}) = 3 \tag{3.4}
$$
This will create a contradiction that $0 = 3$ or $\infty = 3$. Such contradiction is because, the Vitali sets are not measurable.
