    更新日志 1.3.X

    版本 1.3.0

    版本 1.3.0 - 新功能、重构、主要改进和用户体验优化。

    🐛 Bug 修复：

    • 修复 Bug #70，自动编号行为现已修复。

    ⭐ 新功能：

    • 全新的设置标签页，具有分组设置和子面板，使设置比以前更方便。不再有烦人的滚动。
    • 图表引用（来自 Alicecomma 的 功能 #7）现已支持，包括标题、编号和描述。
    • 标注引用，我们可以将表格添加到标注中并对其进行引用。此功能还为您提供了引用表格、文本片段甚至定理的灵活性（引用标注中的所有表格以进行引用）。
    • 为图表和标注添加 PDF 导出支持。
    • 添加公式管理面板，允许您检索特定公式的所有引用，并跳转到它们的位置
      • 大纲视图、筛选和折叠：支持列表视图和大纲视图。
      • 拖放引用：现在可以从公式管理面板拖放公式来进行引用。
      • 支持通过拖放进行跨文件引用
      • 拖放无标签公式时提示
      • 切换文件和编辑器中输入时动态更新面板
      • 交互式：支持双击跳转和 Ctrl + 双击（新面板+跳转）

    🏗️ 重构：

    • 对于设置部分，重构代码并将其拆分为多个模块。

    🚀 增强和改进：

    • 添加 增强 #50，自动编号公式后重新定位光标的更好行为。
    • 跳转到另一个面板中的公式时，现在检查是否已存在该文件的面板，如果存在则在该面板中跳转。
    • 调试模式配置现在在重启 Obsidian 后会保留，因此无需每次都切换调试模式。
    • 重新制作全新且更简洁的教程。
    • 重新制作 Readme 文件，使其更友好和简洁。同时添加中文翻译版 Readme。

    版本 1.3.3

    版本 1.3.3 - Bug 修复、新功能和增强。

    重要更新摘要：

    • 图表引用预览现已支持 Excalidraw！
    • 图表和标注引用的自动补全现已可用！
    • 图表的自动编号和标签重命名现已可用！

    设置更新：

    • 为清晰起见，我们将变量名 quoteCitationPrefixes 重构为 calloutCitationPrefixes。因此您当前的标注引用前缀设置将被恢复。请务必在设置标签页中重新设置「标注引用前缀」以使标注引用正常工作。

    ⭐ 新功能：

    • 图表引用和标注引用的自动补全。
      • 自动补全支持 2 种模式：富模式和紧凑模式（查看 设置 > 高级 > 自动补全中显示完整预览）。
      • 对于紧凑模式，悬停在自动补全项上将显示图表或标注的预览。（查看 设置 > 高级 > 悬停自动补全项时显示预览）
    • 图表标签重命名。
    • 图表标签自动编号。
      • 自动编号或重命名标签后，相关图表引用应随之更新。
    • 添加功能 #140，公式管理面板中的框选公式筛选。
      • 框选当前公式的命令。
    • 添加功能 #143，图表引用现在支持 Excalidraw。
    • 图表引用格式现在可以引用区域预览。
    • 添加移动设备支持，同时通过了安卓设备测试。

    🐛 Bug 修复：

    • 修复 Bug #136，为标注中的多行公式添加标签现在正常工作。
    • 修复故障 #137，当公式面板文件和当前活动文件不同时，将创建新窗口并跳转到正确位置。
    • 修复 Bug #135，在一些折叠标题之前的位置编辑文件时，现在不会展开。
    • 修复 Bug #141，嵌套标注现在可以在标注引用预览中正确显示。
    • 修复 Bug #142，标注中的图表现在可以在预览中被正确识别和渲染。

    🚀 增强：

    • 修复设置标签页中重新排序按钮的位置样式。

    • 添加命令「插入图表引用」以便快速插入图表引用。

    • 悬停在标注和图表的引用上标上现在会显示文件链接。

    • ❌ 停止支持网页链接图表的标题，并在快速入门教程中添加了不支持的原因。
